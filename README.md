## 🚀 開發初衷：解決試算痛點 (The Gap)

在台灣，納稅人若想預估稅額，通常面臨兩個尷尬的選擇：

1. **官方網頁試算 (eTax)**：僅供基礎參考。對於「夫妻合併申報」支援不全，缺乏方案自動優化比較，且介面過長，需要捲動，操作不便。金額欄位又無千分位顯示，容易輸入錯誤。
2. **官方正式報稅軟體**：功能雖完整且精確，但**每年僅在 5 月報稅期間開放使用**，且操作需要驗證登入，不能快速試算。非報稅期間若想提早佈局財務，或想回算往年稅金，幾乎求助無門。

**本專案的定位：**
這是一個 **「全時段」**、**「全方案」** 且 **「高效率」** 的專業試算工具。你不需要等到五月，也不需要忍受官方工具的限制，一個畫面就能精準掌握 110 至 114 年度的綜所稅。

> **📌 重要提醒：**
> 本試算器支援之 **114 年度 (2025)** 稅務參數，即為 **115 年 (2026) 報稅期間** 所適用之計算邏輯。

## ✨ 職人級 UX 特色 | Advanced UX Features

### 1. 財務級輸入體驗 (Financial-Grade Input)
* **千分位即時顯示**：輸入金額時自動呈現千分位符號（如 `1,000,000`），有效防止因「0」過多而導致的輸入錯誤。這是目前官方網頁版所欠缺的細節。
* **內建加號運算**：金額欄位支援直接進行「加法運算」。例如在本人其他欄輸入 `30000+15000`，系統會自動結算為 `45,000`。免去切換計算機的困擾，適合彙整多筆小額收入。

### 2. 跨年度回算支援 (Historical Retrospective)
* **獨家功能**：官方軟體過期即關閉，本工具完整收錄 **110 ~ 114(2021~2025) 年度** 的稅務參數。讓你隨時核對過去五年的稅務狀態，或針對往年收入進行補申報前的精準預算。

### 3. 全案自動優化 (Scenario Comparison)
傳統試算只能手動切換方案，我們幫你 **一次跑完五大申報路徑**，直接導航至 **Best Choice**：
* 全戶合併計稅 / 配偶薪資分開 / 本人薪資分開 / 配偶各類分開 / 本人各類分開。
* **股利自動判定**：同步試算「8.5% 抵減」與「28% 分離計稅」，自動選擇最低稅額。

## 🛠️ 技術指標 | Technical Specifications

* **單頁戰術看板**：數值填入，結果秒出。支援**「已扣繳稅額」**填寫，直接預覽退稅額。
* **法規細節對齊**：
    * 免稅額、標準扣除額、薪資特別扣除、基本生活費差額。
    * **特別扣除額**：幼兒學前、長期照顧、身心障礙、教育學費。
    * **113年新制**：房租特別扣除額與排富條款判定。
* **極致適應與隱私**：支援高解析度縮放。純 **JS+HTML+CSS** 鑄造，所有數據離線運行，數據不出門，確保財務隱私。
